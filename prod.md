# SSH Monitor

> 다중 SSH 연결 및 터미널 분할 기능을 제공하는 웹 기반 모니터링 도구

---

## 1. 개요

SSH 연결 관리, 단축키 지원, 멀티 터미널 화면을 제공하는 웹 모니터링 애플리케이션

---

## 2. 레이아웃

| 영역 | 비율 | 용도 |
|------|------|------|
| Left Sidebar | 2 | 연결 목록 / 관리 |
| Main Terminal | 7 | 터미널 화면 (분할 지원) |
| Right Panel | 1 | 상태 정보 / 설정 |

---

## 3. 주요 기능

### 3.1 연결 관리
- **추가**: SSH 연결 정보 등록 (호스트, 포트, 인증 방식)
- **연결**: 등록된 서버에 SSH 접속
- **삭제**: 연결 정보 제거
- **편집**: 기존 연결 정보 수정

### 3.2 터미널 화면
- **다중 연결**: 하나의 화면에서 여러 SSH 세션 동시 운영
- **화면 분할**: 1분할 / 2분할 / 4분할 지원
- **동적 연결 변경**: 각 분할 영역에서 독립적으로 연결 전환 가능

### 3.3 단축키
- 분할 전환, 연결 선택, 터미널 포커스 등 키보드 조작 지원

---

## 4. 기술 스펙

### 4.1 아키텍처
```
┌─────────────────────────────────────────┐
│              Docker Container           │
├──────────────────┬──────────────────────┤
│   Frontend       │      Backend         │
│   (Vue 3)        │      (Java/Spring)   │
├──────────────────┴──────────────────────┤
│         Local Storage (연결 정보)        │
└─────────────────────────────────────────┘
```

### 4.2 기술 스택

| 구분 | 기술 |
|------|------|
| Frontend | Vue 3, TypeScript, xterm.js |
| Backend | Java, Spring Boot, JSch |
| Infra | Docker, Docker Compose |
| Storage | 브라우저 LocalStorage (연결 정보) |

### 4.3 설계 원칙
- **구조적 독립**: Docker 컨테이너로 환경 분리
- **보안**: 연결 정보는 클라이언트 로컬에서만 관리 (서버 미저장)
- **확장성**: 모듈화된 컴포넌트 구조

## 5. 개발 로드맵

### 📦 1차 개발: MVP (기본 기능)

> 목표: 단일 SSH 연결이 가능한 최소 동작 버전

| 구분 | 작업 항목 |
|------|----------|
| **환경 구성** | Docker Compose 설정, 프로젝트 구조 생성 |
| **Backend** | Spring Boot 프로젝트 초기화, SSH 연결 API (JSch) |
| **Frontend** | Vue 3 프로젝트 초기화, 기본 레이아웃 (2/7/1) |
| **터미널** | xterm.js 연동, WebSocket 통신 |
| **연결 관리** | 연결 추가/삭제 (LocalStorage 저장) |

**산출물:**
- 단일 터미널에서 SSH 연결/명령 실행 가능
- 연결 정보 저장 및 재사용

---

### 🔀 2차 개발: 멀티 터미널

> 목표: 화면 분할 및 다중 연결 지원

| 구분 | 작업 항목 |
|------|----------|
| **화면 분할** | 1분할 / 2분할 / 4분할 레이아웃 구현 |
| **다중 세션** | 분할 영역별 독립 SSH 세션 관리 |
| **연결 전환** | 각 영역에서 연결 선택/변경 기능 |
| **상태 표시** | Right Panel에 연결 상태 정보 표시 |
| **연결 편집** | 기존 연결 정보 수정 기능 |

**산출물:**
- 최대 4개 터미널 동시 운영
- 각 터미널 독립적 연결 관리

---

### ⚡ 3차 개발: 고급 기능

> 목표: 사용성 개선 및 확장 기능

| 구분 | 작업 항목 |
|------|----------|
| **단축키** | 분할 전환 (`Ctrl+1~4`), 포커스 이동, 빠른 연결 |
| **인증 확장** | SSH 키 파일 업로드 지원 |
| **연결 그룹** | 폴더/태그 기반 연결 분류 |
| **세션 관리** | 명령 히스토리, 세션 로깅 |
| **UI/UX** | 다크/라이트 테마, 터미널 폰트 설정 |

**산출물:**
- 키보드 중심 조작 가능
- 대규모 서버 관리에 적합한 구조
